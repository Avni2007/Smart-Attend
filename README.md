Smart Attend 🎓📸
AI-Powered Smart Attendance Management System

Smart Attend is an intelligent attendance management system that automates attendance tracking using modern technologies such as Computer Vision, Face Recognition, and Flask Web Development.
The project is designed to reduce manual effort, improve accuracy, and provide a seamless attendance experience for educational institutions and organizations.

🚀 Features
🎯 Real-time Face Detection & Recognition
🧠 AI-Based Attendance Automation
📷 Webcam Integration
📊 Automatic Attendance Logging
🌐 Flask Web Application
📁 CSV/Database Attendance Storage
⚡ Fast & Accurate Recognition
💻 Clean and Responsive User Interface
🛠️ Tech Stack
Technology	Usage
Python	Core Development
Flask	Backend Framework
OpenCV	Face Detection
face_recognition	Facial Recognition
HTML5 / CSS3 / JavaScript	Frontend UI
NumPy / Pandas	Data Handling
CSV / Database	Attendance Records

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Avni2007/Smart-Attend.git
2️⃣ Navigate to the Project Directory
cd Smart-Attend
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Run the Application
python app.py
🌐 Usage

After running the server, open your browser and visit:

http://127.0.0.1:5000/

The system will start detecting and recognizing faces through the webcam and automatically mark attendance.
