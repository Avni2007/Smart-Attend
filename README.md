# Smart Attend 🎓📸

### AI-Powered Smart Attendance Management System

Smart Attend is an intelligent attendance management system that automates attendance tracking using **Face Recognition** and **Liveness Detection** technologies.

The system is designed to eliminate proxy attendance, improve accuracy, reduce manual effort, and provide a secure and efficient solution for educational institutions and organizations.

---

## 📌 Project Overview

Traditional attendance systems such as manual registers, RFID cards, and fingerprint scanners often face issues like:

* Proxy attendance
* Time-consuming manual processes
* Hardware dependency
* Inaccurate attendance records

Smart Attend solves these problems by leveraging **Artificial Intelligence**, **Computer Vision**, and **Real-Time Face Recognition**.

The system detects and verifies a person's face, checks liveness to prevent spoofing attacks using photos/videos, and automatically marks attendance.

## 🚀 How It Works

1. Register user face images
2. Start the attendance system
3. Camera detects faces in real time
4. Liveness detection verifies authenticity
5. Attendance is marked automatically
6. Attendance records are stored securely
